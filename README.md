An AI-powered web application that compares resumes with job descriptions, calculates ATS scores, identifies matching and missing skills, provides AI-based improvement suggestions, and generates downloadable PDF reports.

## 🚀 Live Demo
https://ai-resume-screening-8hjajttosvkrgxfetdp8ar.streamlit.app/

## 💻 Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP
- Plotly
- PyPDF2
- python-docx
- ReportLab

## ✨ Features
- Upload Resume (PDF, DOCX, TXT)
- Upload or Paste Job Description
- Resume Match Analysis
- ATS Score Calculation
- Matching & Missing Skills Detection
- AI Resume Improvement Tips
- ATS Score Breakdown Chart
- Download Professional PDF Report
